# Arduino Kit 1 - Reference Code

This repository contains starter code and reference examples for learning:
- Arduino basics
- WS2812B LEDs
- RFID (RC522)
- Hall effect sensors

These examples can be used to complete tasks and provide a basic code structure for beginners.

Contents include:
- modified examples from official documentation
- personal experiments and edits
- beginner-friendly starter templates
- AI-assisted reference code
